_featured_tags/hyde, hydejack 
	- menu : true이면 메뉴로 형성되는 듯.
	- title : 메뉴(네비게이션 바)에 노출 되는 이름.
	- slug : posts들의 태그 이름과 매칭되는 이름.

_posts
	- [tag] : 이 태그 이름 별로 위의 _featured_tags 메뉴에 매칭됨.
	- layout : 포스트의 타입(종류). _layouts에 있는 애들을 끌고 오는 듯.


layout 차이 : 
	- post : 작성자, 작성일시 나옴.
	- page : 작성자, 작성일시 안나옴.
	- blog : 왜인지 모르겠는데 안뜸...
